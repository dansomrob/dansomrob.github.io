Kinematics

A particle is given an ipulse giving an inital velocity of \(x_0\). It then has an acceleration given by \(a=-bv\) where \(b\) is a constant.
Obtain an expression for \(v(t)\).

So, let's begin by writing the equation for acceleration as \[\frac{dv}{dt}=-bv\] If we divide by \(v\) and multiply by \(dt\) we get \[\frac{dv}{v}=-bdt\] Now, if we intergrate both sides we get \[\int {\frac{dv}{v}}=\int {-b\,dt}\] \[ln\,v=-bt\] And then raising everything to the power of \(e\) \[v=v_0e^{-bt}\] Boom! Solved! This is known as separation of variables. Kinda important.

Force and Potential

It's important to note a key relationship between the potential of a particle and the force it experiences. \[F=-\frac{\partial V}{\partial x}\] Beautiful.

Fermat's Principle

Fermat's priciple states that

"Nature operates by means and ways that are easiest and fastest."

Or, written more simply

Nature is lazy

This simple idea will allow us to derive equations of motion for any system (including quantum!).

Let's look at a good example of this - Snell's Law.

The Lagrangian

\[L(x,v)=T(v)-V(x)\] Where \(T(v)\) is the kinetic energy and \(v(x)\) is the potential energy.
Therefore, for a one-dimensional partical when \(V(x)=mgx\) is \[L(x,v)=\frac{1}{2}mv^2-mgx\]

Action

Now that I've told you a new idea, how about another one?
Mathematically definded as: \[\mathcal{A}=\int_{t_0}^{t_1} L\,dt\]

The Euler-Lagrange Equation

Let's derive the priciple of least action.
Let's start from the equation for action. \[\mathcal{A}=\int_{t_0}^{t_1} L\,dt\] Next, we need to use the calculus of variations - by adding a very small value to the position and velocity. \[Action=\int_{t_0}^{t_1} ùêø((x+h),(v+\dot{h}))\] Then using the Taylor Expansion we can find that: \[Action=\int_{t_0}^{t_1} ùêø(x,v)+\frac{h}{1!}\frac{\partial L}{\partial x}+\frac{\dot{h}}{1!}\frac{\partial L}{\partial v}+ ...\] We can keep on going for higher powers but this isn't important.

This is because when \(h\lt\lt1\) we find that \(h^1\lt\lt h^2\) and \(h^1\) is bigger than all higher powers of \(h\)

Therfore, the first order terms of the expansion are equal to zero. \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}+\dot{h}\frac{\partial L}{\partial v}\] \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}+\frac{dh}{ht}\frac{\partial L}{\partial v}\] Now we can use intergration by parts to simplify the second part of the above equation. \[\int_{t_0}^{t_1} \frac{dh}{dt}\frac{\partial L}{\partial v}=h\frac{\partial L}{\partial v}-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] For the path of least action, h will tend to zero. Therefore we can say that \[\lim_{h\to 0} h\frac{\partial L}{\partial v} = 0\] This means our intergration by parts is equal to \[\int_{t_0}^{t_1} \frac{dh}{dt}\frac{\partial L}{\partial v}=-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] Subbing this we find that \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] If we factorise out the h \[0=\int_{t_0}^{t_1} h\biggl(\frac{\partial L}{\partial x}-\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\biggr)\] It's important to note that h can be any value. Therefore for the above to be true we have to assume that \[\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)-\frac{\partial L}{\partial x}=0\] And this is the Euler-Lagrange equation. Woohoo!

Pendulumn



We'll begin by asuming we'll be using polar cordinates for this example (the euler-lagrange equation works
in both cartesian and polar coordinates, see later). Let's also note that \(v(x)=mgh\) and \(T(v)=\frac{1}{2}mv^{2}\)
Therefore we need to trasform the height and velocity into polar cordinates. \[h=l-lcos\theta\] \[v=l\omega\] As \(l\) is constant this works. Note that for if we let \(v(x)=mgh\) Then the lagrangian equals: \[L=\frac{1}{2}mv^{2}-mgh\] If we sub in our new equations for h and x into our lagrangian we get: \[L=\frac{1}{2}m\omega^{2}l^{2}-mgl+mglcos\theta\] Then using the euler-lagrange equation we find that \[m\dot{\omega}l^{2}+mglsin\theta=0\] Dividing by m and l \[\dot{\omega}l+gsin\theta=0\] Cheeky rearrange \[\dot{\omega}=\frac{gsin\theta}{l}\] And if we approached this from a newtonian perspective we would get the same answer.
