Hamiltonian mechanics occurs in phase space.

Phase space is when we plot the position and velocity of the system. Genreally we'll use a 2d graph to describe
a single particle in 1d (ie. 1d positon and 1d velocity).
In Hamiltonian we use equations to determine the movenemt of the state vector through the phase space. Simple! \[\frac{d}{dt}F(x,v)=\{F(x,v),H\}=\frac{\partial F}{\partial x}\frac{\partial H}{\partial p}-\frac{\partial F}{\partial p}\frac{\partial H}{\partial x}\] This is the hamiltonian equation - but actually is it?
