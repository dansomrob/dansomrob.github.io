Wavevector

If we look at \(\sin {kx}\) we know that \(x\) has units of \(m\). Therfore, doing a dimensional
analysis we see that \(k\) must have units of \(m^{-1}\). \(k\) is called the wavevector.
We define the wavevector as \[k=\frac{2\pi}{\lambda}\]



From this diagram we can see that \[n\times \frac{\lambda}{2}=a \Rightarrow \lambda=\frac{2a}{n}\] Which menas that \[k=\frac{2\pi}{\lambda}=\frac{n\pi}{a}\] Therefore \(\lambda\) can only take specific values. This is quantisation!
