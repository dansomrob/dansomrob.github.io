The Action of A System

Let's imagine a particle moving from point A to point B, if we cacluated the lagrangian at every time step and sum them we get the Action of the system. \[\mathcal{A}=\int_{t_0}^{t_1} Ldt\tag{1.2}\] Or, written a bit nicer (and assuming \(V(x)=mgx\)): \[Action=\int_{t_0}^{t_1} \frac{1}{2}mv^2 - mgx\tag{1.3}\]

The Derivation of the Euler-Lagrange Equtaion

We need to use the calculus of variations, we do this by adding a very small value to the position and velocity. \[Action=\int_{t_0}^{t_1} ùêø((x+h),(v+\dot{h}))\] Then using the Taylor Expansion we can find that: \[Action=\int_{t_0}^{t_1} ùêø(x,v)+\frac{h}{1!}\frac{\partial L}{\partial x}+\frac{\dot{h}}{1!}\frac{\partial L}{\partial v}+...\] We can keep on going for higher powers but this isn't important. The only part which is important is the first order, which will be equal to zero. \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}+\dot{h}\frac{\partial L}{\partial v}\] \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}+\frac{dh}{ht}\frac{\partial L}{\partial v}\] Now we can use intergration by parts to simplify the second part of the above equation. \[\int_{t_0}^{t_1} \frac{dh}{dt}\frac{\partial L}{\partial v}=h\frac{\partial L}{\partial v}-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] For the path of least action, h will tend to zero. Therefore we can say that \[\lim_{h\to 0} h\frac{\partial L}{\partial v} = 0\] This means our intergration by parts is equal to \[\int_{t_0}^{t_1} \frac{dh}{dt}\frac{\partial L}{\partial v}=-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] Subbing this we find that \[0=\int_{t_0}^{t_1} h\frac{\partial L}{\partial x}-\int_{t_0}^{t_1} h\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\] If we factorise out the h \[0=\int_{t_0}^{t_1} h\biggl(\frac{\partial L}{\partial x}-\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)\biggr)\] It's important to note that h can be any value. Therefore for the above to be true we have to assume that \[\frac{d}{dt}\biggl(\frac{\partial L}{\partial v}\biggr)-\frac{\partial L}{\partial x}=0\tag{1.4}\] And this is the Euler-Lagrange equation. Woohoo!
