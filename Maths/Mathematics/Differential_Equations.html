Differentatial Equations

Differential equations are found in the form \[\frac{dy}{dx}=f(x)\] This is an ordinary differential equation. A partial differental equation looks like \[\frac{\partial y}{\partial x}=f(x)\] What we have to learn is how to solve them. Both equations i've shown so far are first order, we'll look at them first.

Let's beging by looking at a nice example, nuclear decay.

Example 1

\[\frac{dN}{dt}=\lambda N\] In A-Level we substitute \(\frac{dN}{dt}\) as activity \(A\).

Now lets solve this equation. We'll begin by using separtion of variables. \[dN=\lambda N\,dt\] \[\frac{dN}{N}=\lambda \,dt\] Now if we intergrate both sides \[\int {\frac{dN}{N}}=\int {\lambda\,dt}\] \[ln|N|=\lambda t+c\] Then raising everything to \(e\) \[N=e^{\lambda t}e^c\] Now if we say \(e^c=N_0\) then we get \[N=N_0e^{-\lambda t}\] Graphically this equation is
